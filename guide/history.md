# 历史记录

OCR Pro 会自动保存识别和翻译的历史记录，方便回顾和重用。

## 查看历史

1. 呼出 uTools
2. 输入 `历史记录`
3. 浏览历史列表

## 历史记录内容

每条记录包含：

- **时间**：操作时间
- **类型**：OCR / 翻译
- **内容**：识别或翻译的文本
- **原图**：OCR 的原始图片（如果有）

## 历史操作

- **复制**：复制历史内容
- **重新翻译**：对历史内容再次翻译
- **删除**：删除单条记录
- **清空**：清空所有历史

## 历史设置

在设置中可以配置：

- **最大保存数量**：控制历史记录上限
- **自动保存**：开启/关闭自动保存

::: tip 存储说明
历史记录存储在本地，不会上传到云端，保护您的隐私。
:::
