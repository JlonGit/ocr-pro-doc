# 图片翻译

OCR Pro 支持直接翻译图片中的文字，保持原图布局。

## 功能特点

- 🖼️ 保持原图布局
- 🌐 多语言支持
- ⚡ 快速处理

## 使用方式

### 截图图片翻译

1. 呼出 uTools
2. 输入 `截图图片翻译`
3. 框选需要翻译的区域
4. 查看翻译结果

### 图片文件翻译

1. 拖拽图片文件到 uTools
2. 选择 `图片翻译`
3. 查看翻译结果

### 剪贴板图片翻译

1. 复制图片到剪贴板
2. 呼出 uTools
3. 输入 `剪切板图片翻译`

## 与 OCR 后翻译的区别

| 特性 | 图片翻译 | OCR 后翻译 |
|------|---------|-----------|
| 输出形式 | 图片 | 文本 |
| 布局保留 | ✅ | ❌ |
| 可编辑性 | ❌ | ✅ |
| 适用场景 | 截图分享 | 文本提取 |

## 使用场景

- 📱 翻译应用界面截图
- 📖 翻译书籍/文章图片
- 🎮 翻译游戏界面
- 📧 翻译图片格式的邮件

## 使用建议

::: tip 获得最佳效果
1. 确保图片文字清晰
2. 避免背景过于复杂
3. 文字区域足够大
:::

::: warning 注意事项
图片翻译需要使用支持此功能的翻译服务，请确保已正确配置。
:::
