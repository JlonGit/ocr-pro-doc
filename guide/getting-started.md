# 快速开始

欢迎使用 OCR Pro！本指南将帮助你快速上手这款强大的 OCR 文字识别插件。

## 前置条件

在开始之前，请确保你已经安装了 [uTools](https://www.u.tools/)。

## 安装插件

1. 打开 uTools
2. 搜索 `OCR Pro`
3. 点击获取

<img src="../image/OCR Pro.png" alt="OCR Pro" style="zoom:80%;" />

## 基本使用

### 截图识别

最常用的 OCR 方式：

1. 呼出 uTools（默认 `Alt + Space`）
2. 输入 `截图文字识别`

3. 框选需要识别的区域
4. 查看识别结果

<img src="../image/ocr.png" alt="OCR" style="zoom:40%;" />

::: tip 快捷操作
可以为该 `功能指令` 设置全局快捷键，实现一键截图识别。
:::

### 图片文件识别

拖拽图片文件到 uTools 输入框，或者点击 OCR 页面的 `选择` 按钮即可触发识别。

### 剪贴板识别

1. 复制图片到剪贴板
2. 呼出 uTools 超级面板即可自动匹配识别
