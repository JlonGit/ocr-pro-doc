# 快速开始

欢迎使用 OCR Pro！本指南将帮助你快速上手这款强大的 OCR 文字识别插件。

## 前置条件

在开始之前，请确保你已经安装了 [uTools](https://www.u.tools/)。

## 安装插件

1. 打开 uTools
2. 输入 `插件中心`
3. 搜索 `OCR Pro`
4. 点击安装

## 基本使用

### 截图识别

最常用的 OCR 方式：

1. 呼出 uTools（默认 `Alt + Space`）
2. 输入 `截图文字识别`
3. 框选需要识别的区域
4. 查看识别结果

::: tip 快捷操作
可以为该功能设置全局快捷键，实现一键截图识别。
:::

### 图片文件识别

拖拽图片文件到 uTools 输入框即可触发识别。

### 剪贴板识别

1. 复制图片到剪贴板
2. 呼出 uTools
3. 输入 `剪切板图片识别`

## 首次配置

首次使用前，建议完成以下配置：

### 1. 配置 OCR 服务

OCR Pro 支持多种 OCR 服务商：

| 服务 | 特点 | 推荐场景 |
|------|------|----------|
| 本地主机 | 无需联网，免费 | 日常文字识别 |
| 百度 OCR | 高精度，功能丰富 | 专业场景 |
| 腾讯 OCR | 速度快，价格低 | 通用场景 |

详细配置请参考 [OCR 服务配置](/config/ocr-services)。

### 2. 配置翻译服务（可选）

如需使用翻译功能，请配置至少一个翻译服务。

详细配置请参考 [翻译服务配置](/config/translate-services)。
