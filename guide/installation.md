# 安装说明

## 系统要求

- **操作系统**：Windows / macOS
- **uTools 版本**：最新版本

## 安装方式

### 方式一：插件中心安装（推荐）

1. 打开 uTools（默认快捷键 `Alt + Space`）
2. 输入 `插件中心`，回车进入
3. 在搜索框输入 `OCR Pro`
4. 找到插件后点击 `安装` 按钮
5. 等待安装完成

### 方式二：本地安装

如果你有插件的 `.upx` 文件：

1. 打开 uTools
2. 输入 `插件中心`
3. 点击右上角设置按钮
4. 选择 `本地安装`
5. 选择 `.upx` 文件

## 安装后配置

安装完成后，建议进行以下配置：

### 1. 配置 OCR 服务

插件支持多种 OCR 服务，首次使用需要配置：

```
uTools 输入 → 设置 → OCR 模型服务
```

选择适合你的服务并完成 API 配置。

### 2. 设置全局快捷键

为常用功能设置快捷键可以大幅提升效率：

1. 打开 uTools 设置
2. 进入「全局快捷键」
3. 为以下功能绑定快捷键：
   - 截图文字识别
   - 静默OCR识别
   - 翻译

::: tip 推荐快捷键设置
- `Ctrl + Alt + O`：截图文字识别
- `Ctrl + Alt + S`：静默 OCR（识别并自动复制）
- `Ctrl + Alt + T`：翻译
:::

## 常见问题

### 插件无法安装

- 确保 uTools 是最新版本
- 检查网络连接
- 尝试重启 uTools 后重新安装

### 安装后无法使用

- 检查是否正确配置了 OCR 服务
- 对于需要 API Key 的服务，确认 Key 是否有效
