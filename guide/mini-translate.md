# 小窗翻译

小窗翻译是一个独立的悬浮翻译窗口，可以在使用其他应用时同时进行翻译。

## 功能特点

- 🪟 独立悬浮窗口
- 📍 位置记忆功能
- ⌨️ 快捷键支持
- 🔄 自动翻译选项

## 使用方式

### 打开小窗翻译

1. 呼出 uTools
2. 输入 `小窗翻译`
3. 弹出独立悬浮窗口

### 翻译剪贴板文本

1. 复制需要翻译的文本
2. 呼出 uTools
3. 输入 `翻译复制的文本-小窗`

## 配置选项

在设置中可以配置小窗翻译：

### 显示设置
- **显示标题栏**：是否显示窗口标题栏
- **窗口置顶**：窗口始终在最前端

### 翻译设置
- **结束编辑后自动翻译**：失去焦点时自动翻译
- **即时自动翻译**：输入时实时翻译

### 复制设置
- **复制后自动关闭**：复制结果后自动关闭窗口
- **默认复制模型一结果**：Ctrl+C 快捷键复制第一个模型的结果

## 快捷操作

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 执行翻译 |
| `Ctrl + C` | 复制结果 |
| `Esc` | 关闭窗口 |
| 双击右键 | 复制结果 |

## 与主窗口的区别

| 特性 | 主翻译窗口 | 小窗翻译 |
|------|-----------|---------|
| 窗口类型 | uTools 内部 | 独立悬浮 |
| 多任务 | 需切换 | 可同时使用 |
| 位置 | 固定 | 可拖动、记忆 |
| 适用场景 | 专注翻译 | 边工作边翻译 |
