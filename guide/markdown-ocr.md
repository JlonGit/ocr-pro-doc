# Markdown 识别

OCR Pro 支持将文档图片智能转换为 Markdown 格式，保留原文档结构。

## 功能特点

- 📄 保留文档结构
- 🏷️ 识别标题层级
- 📋 识别列表格式
- 🔗 识别链接和代码块

## 使用方式

1. 呼出 uTools
2. 输入 `Markdown识别`
3. 截取文档区域
4. 获取 Markdown 格式文本

## 支持识别的元素

| 元素 | 支持情况 |
|------|----------|
| 标题 | ✅ H1-H6 |
| 段落 | ✅ |
| 粗体/斜体 | ✅ |
| 列表 | ✅ 有序/无序 |
| 引用 | ✅ |
| 代码块 | ✅ |
| 表格 | ✅ |
| 分隔线 | ✅ |

## 输出示例

识别类似这样的文档：

```markdown
# 一级标题

这是一段正文内容。

## 二级标题

- 列表项 1
- 列表项 2

> 这是引用内容
```

## 使用场景

- 📚 将书籍页面转为电子笔记
- 📰 将文章截图转为 Markdown
- 📋 将 PDF 内容转为可编辑格式
- 🗒️ 快速记录白板内容

## 使用建议

::: tip 最佳实践
1. 确保文档排版清晰
2. 识别中文文档效果更佳
3. 复杂排版可能需要手动调整
4. 结合表格识别处理表格内容
:::
