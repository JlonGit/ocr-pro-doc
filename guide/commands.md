# 快捷命令

OCR Pro 提供丰富的快捷命令，满足不同场景需求。

## 命令列表

### OCR 识别

| 命令 | 说明 | 触发方式 |
|------|------|----------|
| `OCR Pro` | 打开主界面 | 关键词 |
| `截图文字识别` | 截图 OCR | 关键词 |
| `静默OCR识别` | 截图识别并复制 | 关键词，后台运行 |
| `剪切板图片识别` | 识别剪贴板图片 | 关键词 |
| `图片识别` | 识别图片文件 | 拖拽文件 |

### 翻译

| 命令 | 说明 | 触发方式 |
|------|------|----------|
| `翻译` | 打开翻译界面 | 关键词 |
| `小窗翻译` | 打开悬浮翻译窗口 | 关键词 |
| `翻译复制的文本` | 翻译剪贴板文本 | 关键词 |
| `翻译复制的文本-小窗` | 小窗翻译剪贴板 | 关键词 |
| `文本翻译` | 翻译选中文本 | 选中文本后呼出 |
| `文本翻译-小窗` | 小窗翻译选中文本 | 选中文本后呼出 |

### OCR + 翻译

| 命令 | 说明 | 触发方式 |
|------|------|----------|
| `截图识别并翻译` | 截图 OCR 后翻译 | 关键词 |
| `截图图片翻译` | 截图后图片翻译 | 关键词 |
| `图片识别并翻译` | 图片 OCR 后翻译 | 拖拽文件 |
| `图片翻译` | 翻译图片中文字 | 拖拽文件 |

### 特殊识别

| 命令 | 说明 | 触发方式 |
|------|------|----------|
| `表格识别` | 识别表格内容 | 关键词 |
| `公式识别` | 识别数学公式 | 关键词 |
| `手写公式识别` | 手写公式识别 | 关键词 |
| `Markdown识别` | 识别为 Markdown | 关键词 |

### 其他

| 命令 | 说明 | 触发方式 |
|------|------|----------|
| `设置` | 打开设置页面 | 关键词 |
| `历史记录` | 查看历史 | 关键词 |

## 设置快捷键

建议为常用命令设置全局快捷键：

1. 打开 uTools 设置
2. 进入「全局快捷键」
3. 找到对应命令并设置快捷键

::: tip 推荐配置
| 功能 | 推荐快捷键 |
|------|-----------|
| 截图文字识别 | `Ctrl + Alt + O` |
| 静默OCR识别 | `Ctrl + Alt + S` |
| 翻译 | `Ctrl + Alt + T` |
| 小窗翻译 | `Ctrl + Alt + M` |
:::
