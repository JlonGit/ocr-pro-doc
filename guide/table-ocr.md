# 表格识别

OCR Pro 支持专业的表格识别功能，可以识别图片中的表格并保持其结构。

## 功能特点

- 📊 保持表格结构
- 📋 支持导出为多种格式
- 🎯 高精度识别

## 使用方式

1. 呼出 uTools
2. 输入 `表格识别`
3. 截取包含表格的区域
4. 查看识别结果

## 支持的表格类型

- 有边框表格
- 无边框表格
- 合并单元格表格
- 复杂嵌套表格

## 导出格式

识别完成后可导出为：

- **纯文本**：使用空格/制表符分隔
- **Markdown 表格**：适合文档使用
- **HTML 表格**：可直接嵌入网页

## 使用建议

::: tip 获得最佳识别效果
1. 确保表格边界清晰
2. 避免表格过于倾斜
3. 保持足够的图片分辨率
4. 单元格内容不要太密集
:::

## 服务要求

表格识别需要使用支持表格识别的 OCR 服务：

- 百度 OCR（表格识别 API）
- 部分其他高级 OCR 服务

::: warning 注意
本地 OCR 模型不支持表格结构识别，只能识别表格中的文字。
:::
