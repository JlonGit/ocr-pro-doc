# 其他配置

除了服务配置外，OCR Pro 还提供丰富的个性化设置。

## 基础配置

### OCR 默认模型

配置默认使用的 OCR 模型数量和顺序。

- **模型数量**：1-3 个
- **模型选择**：从已配置的服务中选择

### 翻译默认模型

配置默认使用的翻译模型数量。

## 翻译页面配置

### 自动翻译

| 选项 | 说明 |
|------|------|
| 结束编辑后自动翻译 | 输入框失去焦点时自动翻译 |
| 即时自动翻译 | 输入过程中实时翻译 |

::: warning 注意
建议只开启其中一个选项，同时开启可能导致重复翻译。
:::

### 显示设置

- 显示/隐藏翻译源语言选择
- 显示/隐藏翻译目标语言选择

## 小窗翻译配置

### 窗口设置

- **显示标题栏**：是否显示窗口标题
- **窗口置顶**：窗口始终在最前端

### 复制设置

- **复制后自动关闭**：复制结果后关闭窗口
- **默认复制模型一结果**：Ctrl+C 复制第一个模型结果

### 自动翻译

与翻译页面配置相同。

## 其他配置

### 双击右键复制

开启后可通过双击右键快速复制识别/翻译结果。

### 隐藏窗口时继续识别

开启后，即使隐藏 uTools 窗口也会继续完成识别任务。

## 恢复默认

如需恢复所有设置为默认值：

1. 进入设置页面
2. 找到「恢复默认设置」选项
3. 确认恢复

::: danger 警告
恢复默认设置会清除所有配置，包括 API Key，请谨慎操作。
:::
