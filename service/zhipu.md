# 智谱AI

智谱AI 的 GLM 系列模型，中文理解能力强。

## API 申请

### 申请步骤

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册/登录账号
3. 进入「API密钥」页面
4. 创建新的 API Key
5. 复制并保存密钥

### 配置参数

| 参数 | 说明 |
|------|------|
| API Key | 智谱AI API 密钥 |
| 模型 | 推荐 `glm-4v` (视觉) 或 `glm-4` |

---

## 模型选择

| 模型 | 特点 |
|------|------|
| glm-4v | 视觉多模态模型，支持图像理解 |
| glm-4 | 旗舰语言模型 |
| glm-4-flash | 快速版本，适合高频调用 |

::: tip 免费额度
新用户可获得免费 token 体验。
:::
