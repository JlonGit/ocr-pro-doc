# Anthropic

Anthropic 的 Claude 系列模型，长文本处理能力强。

## API 申请

### 申请步骤

1. 访问 [Anthropic Console](https://console.anthropic.com/)
2. 注册/登录账号
3. 进入「API Keys」页面
4. 创建新的 API Key
5. 复制并保存密钥

### 配置参数

| 参数 | 说明 |
|------|------|
| API Key | Anthropic API 密钥 |
| 模型 | 推荐 `claude-3-5-sonnet-20241022` |

---

## 模型选择

| 模型 | 特点 |
|------|------|
| claude-3-5-sonnet | 综合能力强，性价比高 |
| claude-3-opus | 最强模型，复杂任务首选 |
| claude-3-haiku | 轻量版本，速度快 |

## 下一步

配置完成后，返回插件进行 [OCR 服务配置](/config/ocr-services) 或 [翻译服务配置](/config/translate-services)。
