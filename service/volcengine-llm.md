# 火山引擎 LLM

火山引擎的豆包大模型，字节跳动旗下，国内领先的 LLM 服务。

## API 申请

### 申请步骤

1. 访问 [火山引擎](https://www.volcengine.com/)
2. 注册/登录账号
3. 进入控制台 → 「火山方舟」
4. 开通大模型服务
5. 在「API 密钥」中创建密钥
6. 获取 **API Key**

### 配置参数

| 参数 | 说明 |
|------|------|
| API Key | 火山引擎大模型 API Key |
| 模型 | 推荐使用豆包系列模型 |

---

## 模型选择

| 模型 | 特点 |
|------|------|
| doubao-pro | 旗舰模型，能力最强 |
| doubao-lite | 轻量版本，速度快 |

## 下一步

配置完成后，返回插件进行 [OCR 服务配置](/config/ocr-services) 或 [翻译服务配置](/config/translate-services)。
