# 火山引擎

火山引擎是字节跳动旗下的云服务平台，提供高性能的 OCR 和翻译服务。

## OCR 服务

火山引擎 OCR 服务技术领先，支持多种识别场景。

### 申请步骤

1. 访问 [火山引擎](https://www.volcengine.com/)
2. 注册/登录账号
3. 进入控制台 → 「视觉智能」→「OCR 文字识别」
4. 开通服务
5. 进入「访问密钥」页面
6. 创建并获取 **Access Key ID** 和 **Secret Access Key**

### 配置参数

| 参数 | 说明 |
|------|------|
| Access Key ID | 火山引擎 Access Key ID |
| Secret Access Key | 火山引擎 Secret Access Key |

### 免费额度

新用户可获得一定的免费试用额度，具体以官网为准。

## 翻译服务

火山引擎机器翻译，基于字节跳动强大的翻译技术。

### 申请步骤

1. 访问 [火山引擎](https://www.volcengine.com/)
2. 登录账号
3. 进入控制台 → 「AI 平台」→「机器翻译」
4. 开通服务
5. 使用之前获取的 **Access Key ID** 和 **Secret Access Key**

### 配置参数

| 参数 | 说明 |
|------|------|
| Access Key ID | 火山引擎 Access Key ID |
| Secret Access Key | 火山引擎 Secret Access Key |

## 下一步

配置完成后，返回插件进行 [OCR 服务配置](/config/ocr-services) 或 [翻译服务配置](/config/translate-services)。
