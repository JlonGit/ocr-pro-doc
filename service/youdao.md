# 有道翻译

网易有道智云翻译 API，中英翻译质量好。

## 翻译服务

有道翻译专注于中英翻译，质量优秀。

### 申请步骤

1. 访问 [有道智云](https://ai.youdao.com/)
2. 注册/登录网易账号
3. 创建应用
4. 开通「文本翻译」服务
5. 获取 **应用 ID** 和 **应用密钥**

### 配置参数

| 参数 | 说明 |
|------|------|
| 应用 ID | 有道智云应用 ID |
| 应用密钥 | 有道智云应用密钥 |

### 免费额度

新用户赠送 100 元体验金。

## 下一步

配置完成后，返回插件进行 [传统模型服务配置](/config/traditional-models)。
